Slatewave for Ghostty mirrors the `terminal.ansi*` block from the VSCode
Slatewave theme, so `ls --color`, `git diff`, and any 256-color TUI read
identically whether you're in the editor or the terminal. Works
cleanly with the `slatewave-omp` prompt on top.
