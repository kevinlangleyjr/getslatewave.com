Slatewave for LSD recolors the column roles LSD already exposes —
owner, group, permissions, size, date, git status, tree edges — so the
row reads spatially: teal for read, amber for write, rose for exec;
teal-300 for hour-old dates fading to slate-400 for older ones; slate
for small files warming to amber for large.

File-type colors (directories, symlinks, archives, media) are driven
by `LS_COLORS`, so Slatewave composes with whatever `LS_COLORS`
generator you already use — [vivid](https://github.com/sharkdp/vivid),
trapd00r/LS_COLORS, or the default. The theme tunes against LSD 1.0+,
which accepts `#rrggbb` hex alongside the legacy named colors and ANSI
256 indices.
