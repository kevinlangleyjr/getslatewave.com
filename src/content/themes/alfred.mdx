Slatewave for Alfred is a single `.alfredappearance` preset — every
slot in Alfred 5's schema gets a semantically-matched Slatewave color,
not just the window background. The selected result title picks up the
teal signature, while unselected titles sit in slate-300 with
slate-500 subtext so nothing competes for attention.

The window uses slate at 90% opacity with a 20pt blur, so Alfred feels
like a frosted pane of the Slatewave editor hovering above your
desktop. ARGB color values map directly to VSCode's `list.*` / `editor.*`
analogues, keeping Alfred's row states aligned with the editor's
file-tree states.
