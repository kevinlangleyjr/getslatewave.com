Slatewave for Sublime Text ships both a color scheme and a UI theme so
every pixel — from editor tokens to the status bar, quick panel, and
auto-complete kind labels — reads in the Slatewave palette. The color
scheme works standalone if you want to keep a different UI theme.

A bundled `Slatewave: Activate Color Scheme & Theme` palette command
flips both at once, so you don't have to set them separately.

A Package Control listing is in flight; until then, the clone path
above is the install.
