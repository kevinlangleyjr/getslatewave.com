Slatewave for bat is a Sublime Text–compatible `.tmTheme` file — the
format `bat` consumes via [syntect](https://github.com/trishume/syntect).
Scope mapping is identical to the [Sublime Text port](/themes/sublime-text/),
so a Go, TypeScript, or Markdown file that highlights a certain way in
your editor highlights the same way when piped through `bat` in the
terminal.

Once registered with `bat cache --build`, the theme cascades to every
tool that delegates highlighting to bat: [git-delta](https://github.com/dandavison/delta)
for pretty diffs, `fzf --preview` for file previews, and
[ripgrep-all](https://github.com/phiresky/ripgrep-all) when piped
through a pager. Set it once, get it everywhere.

Requires a true-color terminal — at 256 colors the slate ramp collapses
and the accent hues drift. Pair with any of the Slatewave terminal ports
(Alacritty, Ghostty, iTerm2, WezTerm, Windows Terminal) for the matched
chrome.
